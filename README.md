# airport-sim

This project simulates the gate operations of a regional airport, including handling of arrivals, departures,
and finite resources such as gates, ground crew, and service vehicles. It optimizes usage time and
make decisions based on random delays.

## Simulation Models
The simulation utilizes stochastic modeling to replicate real-world airport variability:
- Arrivals: Modeled using an Exponential Distribution to simulate a Poisson arrival process.
- Service Times: Uses Triangular Distribution to define realistic minimum, maximum, and most-likely turnaround times.
- Delays: Uses Weibull Distribution to model stochastic environmental interruptions, such as weather-related ground delays.
- Resource Allocation: Uses SJF to serve aircraft with the shortest estimated service time, making throughput more efficient.

## Project Structure
- `main.py`: The entry point for the simulation. It reads scenario parameters from `scenarios.json` and executes the simulation loop.
- `analysis.py`: Performs post-simulation analysis, calculates metrics, and generates trend visualizations.
- `scenarios.json`: External configuration file defining the 10 distinct simulation scenarios (varying arrival rates, gate counts, crew, and vehicle counts).
- `simulation_metrics.csv`: The output file generated by `main.py` containing raw performance data for all runs.
- `run_trends_chart.png`: Visual output generated by `analysis.py` illustrating wait vs. turnaround times.
